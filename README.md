# Ayudemos al Prójimo - Equipo 1

## Descripción

Este proyecto es un sitio web estático desarrollado como parte de un proyecto solidario del Colegio Intelecto. El objetivo es apoyar a comunidades vulnerables, específicamente La Tinajita, a través de información, donaciones y concienciación.

El sitio web promueve la ayuda al prójimo mediante secciones informativas, una galería de imágenes, estadísticas y un formulario de donaciones.

## Características

- **Inicio**: Página principal con introducción al proyecto y llamada a la acción.
- **Sobre**: Información detallada sobre el proyecto y sus objetivos.
- **Ayudar**: Guías y formas de contribuir al proyecto.
- **Galería**: Colección de imágenes relacionadas con el proyecto.
- **Estadísticas**: Datos y métricas sobre el impacto del proyecto.
- **Donar**: Formulario para realizar donaciones.

## Tecnologías Utilizadas

- **HTML5**: Estructura del sitio web.
- **CSS3**: Estilos y diseño responsivo.
- **JavaScript**: Funcionalidades interactivas, como el menú móvil.
- **Bootstrap**: Utilizado parcialmente para layouts flexibles.

## Instalación y Uso

Este es un sitio web estático, por lo que no requiere instalación de dependencias complejas.

1. Clona el repositorio:
   ```
   git clone https://github.com/tu-usuario/ayudemos-al-projimo-equipo-1.git
   ```

2. Navega al directorio del proyecto:
   ```
   cd ayudemos-al-projimo-equipo-1
   ```

3. Abre el archivo `index.html` en tu navegador web preferido.
